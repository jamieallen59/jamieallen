Personal website
=================

Built using Github Pages.

###To run
```shell
npm i
npm run start
```

### To update
Simply add context into `src/data.json`, then add referenced images to `src/images`. That's it!

###Other info
Built using:
  - React
  - Redux
  - JavaScript
  - Webpack
  - LESS
